# Release Notes - Version 3.3.0

This release introduces quick add functionality to product highlights, a new blog layout, and includes fixes across slideshow, header, cart, and quick add components.

## What's Changed

### Added

- [Layered slideshow] Added drop shadow setting
- [Product highlight] Added quick add functionality
- [Cart] Added empty state transition
- [Blog post] Added new grid layout

### Changed

- [Header] Header menu now uses hamburger style on all touch devices
- [Featured Product] Changed thumbnails' default position to bottom in Featured product section
- [Quick add] Polished the close button focus state and sticky add-to-cart button
- [CSS] Removed unused classes, consolidated selectors, and co-located styles across components

### Fixes and improvements

- [Announcement] Fixed Announcement Bar height calculation on initial page load
- [Buttons] Fixed bold button labels on iOS mobile devices
- [Carousel] Fixed quick add button hover jitter
- [Cart bubble] Fixed rendering inconsistencies
- [Cart drawer] Improved cart drawer UI
- [Cart drawer] Fixed cart drawer font size regression
- [Collection] Fixed infinite scroll page crash in iOS
- [Collection links] Fixed collection links style reset after hover
- [Featured product] Fixed accelerated checkout buttons not showing on first load
- [Filters] Fixed price filtering support for zero-decimal currencies
- [Forms] Fixed contact form validation errors appearing in email signup
- [Gift Card] Fixed gift card recipient text area styling
- [Header] Fixed collection images not stretching to full width in mobile drawer
- [Header] Fixed scrolling issue for accordion menus
- [Header] Improved header menu blocks performance
- [Hero section] Fixed accordion clickability
- [Hero section] Fixed image background position in Safari
- [Product media] Improved lightbox view transitions
- [Product pickup] Fixed single-variant product titles
- [Quick add] Fixed modal not closing on mobile and other visual bugs
- [Rich text] Fixed typography presets for nested headings
- [Slideshow] Fixed slide sliver on product card hover
- [Slideshow] Fixed layered slideshow appearing over menu drawer
- [Slideshow] Fixed slideshow content visibility in older browsers
- [Slideshow] Fixed slideshow content disappearing after loop on Safari
- [Styles] Added default radio and checkbox styles
- [Subscriptions] Fixed subscription widget not updating on combined listings variant change
- [Text styles] Fixed font weight override setting sometimes not applying
- [Videos] Fixed aspect ratio for video containers
